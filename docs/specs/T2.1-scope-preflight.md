## 2. Source Data Details
Data has been acquired via the TCGdex API (`v2/id`).
- **Sets Data**: Fetched from `https://api.tcgdex.net/v2/id/sets/[CODE]`
- **Cards Data**: Fetched from `https://api.tcgdex.net/v2/id/cards/[ID]` for every card in the set to capture `category` (supertype) and `types` (element).

## 3. Data Integrity & Inferred Fields
The TCGdex API for the Indonesian language (`id`) currently returns `null` for the `rarity` field. To unblock development, the following inference rules have been applied during CSV generation:

- **Rarity RR**: Assigned to cards with "ex" in the name.
- **Rarity SR**: Assigned to "Secret Rare" cards (localId > official card count).
- **Rarity C**: Assigned as the default for all other cards.
- **Supertype**: Mapped from TCGdex `category`.
- **Element**: Mapped from the first entry in TCGdex `types`.

> [!WARNING]
> **Manual Audit Required**: Due to inferred rarities, the KR4 manual audit (T2.2) is critical. PM and QA must verify a random sample of 30 cards per set to correct rarities before M3 utility goes live.
