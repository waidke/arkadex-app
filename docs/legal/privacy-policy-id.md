# Kebijakan Privasi (Privacy Policy)

**Terakhir Diperbarui: 13 Mei 2026**

Kebijakan Privasi ini disusun berdasarkan **Undang-Undang Nomor 27 Tahun 2022 tentang Pelindungan Data Pribadi (UU PDP)** Republik Indonesia.

## 1. Data yang Kami Kumpulkan
ArkaDex mengumpulkan informasi terbatas untuk menjalankan fungsi aplikasi:
- **Informasi Akun**: Nama, alamat email, dan foto profil (jika Anda mendaftar melalui Google/Apple).
- **Data Koleksi**: Informasi kartu yang Anda tambahkan ke dalam "Digital Binder" Anda.
- **Log Teknis**: Alamat IP, tipe perangkat, dan data penggunaan aplikasi untuk tujuan perbaikan *bug*.

## 2. Tujuan Pemrosesan Data
Kami memproses data Anda untuk:
- Memungkinkan sinkronisasi koleksi Anda di berbagai perangkat.
- Memberikan rekomendasi kartu atau statistik koleksi.
- Menganalisis fitur mana yang paling sering digunakan untuk pengembangan aplikasi lebih lanjut.

## 3. Penyimpanan dan Keamanan
- Data Anda disimpan di server pihak ketiga yang aman (misalnya: Supabase/Firebase) dengan standar enkripsi industri.
- Kami tidak menyimpan informasi pembayaran atau data kartu kredit langsung di server kami.

## 4. Berbagi Data dengan Pihak Ketiga
Kami **TIDAK** menjual data pribadi Anda. Kami hanya berbagi data dengan:
- Penyedia layanan infrastruktur (server/hosting).
- Alat analitik anonim (misalnya: Google Analytics) untuk memahami penggunaan aplikasi secara agregat.

## 5. Hak Anda sebagai Subjek Data (UU PDP)
Sesuai dengan hukum Indonesia, Anda memiliki hak untuk:
1. **Akses**: Mendapatkan informasi tentang pemrosesan data Anda.
2. **Koreksi**: Mengubah data yang tidak akurat (misal: nama profil).
3. **Penghapusan**: Meminta kami untuk menghapus akun dan data koleksi Anda secara permanen.
4. **Penarikan Persetujuan**: Berhenti berlangganan dari notifikasi atau layanan kami.
